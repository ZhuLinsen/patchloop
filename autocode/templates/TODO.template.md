# TODO

This file is a minimal backlog source that PatchLoop can sync into GitHub issues.
Keep long-running execution notes in your own project docs or issue tracker instead
of mixing them into the task source.

使用约定尽量简单：

- 只要写未完成项，用 `- [ ]`
- 一行写一个动作，不必写成长篇背景
- 标题可有可无；想分组就加 `##`

最小可用示例：

- [ ] 修复一个当前最值得优先处理的问题
- [ ] 为关键入口补一条回归测试
- [ ] 优化一个明显影响使用体验的细节

如果你想分组，也可以这样写：

## High Priority

- [ ] 修复行情抓取的一个高频失败场景
- [ ] 为日报生成链路补最小离线测试

## Improvements

- [ ] 清理一个范围明确的历史技术债
- [ ] 改善一个配置或提示文案的易用性
